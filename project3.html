Factors in Lego Pricing

This is an academic project in the class SDS 220. Our group sets up a primary hypothesis about the relationship between a Lego set's recommended price and its number of pieces, and a supplementary one about how Lego themes affect the recommended price of a set.
This is an on-going project. We are at Stage 3: Statistical Plan, where we decides which statistical analysis plan to use for the project. Our group leans towards using linear regression model and a Chi-square test.
The Lego prices dataset is a modified version of a dataset describing Lego sets, originally published by Peterson & Zeigler (2021).
