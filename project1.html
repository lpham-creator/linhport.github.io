IMDB Movie Recommender System

This project aims to mimic popular recommendation engines on popular sites like Hulu, Netflix, HBO, with the goal of using data to build models that can match users with similar film taste and unique ratings together, and identify movies that will suit them.
This will be helpful by gaining users' interest and automatically recommends movies for cohorts of users, cutting the cost of manually choosing movies. The high frequency real-time dataset is scraped from IMDB.com.
One-hot encoding and sparse matrix were implemented in this model and they yielded reliable results.
