Titanic: Machine Learning From Disaster

My first machine learning challenge on Kaggle. After implementing Exploratory Data Analysis on the dataset (including travellers' cabin, fare, sex, and age), I applied one-hot encoding and feature engineering on the data to prepare for my model.
I used a Logistic Regression model, and it acquired 85% accuracy in predicting which traveller survives from the Titanic disaster. This can propel research on supervised learning, and build the bridge to unsupervised learning for larger models.
The Jupyter Notebook for this project can be found here.
