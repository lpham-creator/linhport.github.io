Vegetarian Restaurant Map

This project was built in 24 hours in the timeframe of Technica Hackathon 2022. With a dataset obtained from Datafiniti, I extracted the longitude and latitude of each vegetarian restaurant, and mapped them onto an appropriate CRS. I also assigned the size of the hover tool to the restaurant's average dining expense.
Through this project, I hope to provide customers with some insight so that they can have many dining options. This can be beneficial to vegetarian restaurants as well, as they have a chance to be featured and found by customers.
I coded this project mainly with Python, specifically the pandas package, to manipulate and visualize data.
The Jupyter Notebook for this project can be found here.
